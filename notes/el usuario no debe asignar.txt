El usuario no debe asignar los IDs, eso lo debe asiganr el sistema sin repeticiones que se tengan que valdiar.

La nueva estructura de las tareas será:

"""
[Categoria]
campo1 = ...
campo2 = ...
campo3 = ...
...
"""

Y luego los procesamos y lo dejamos como:
"""
[$id]
categoria = $Categoria
campo1 = ...
campo2 = ...
campo3 = ..
"""

Y así simplemente procesariamos el archivo teporal con los ID que el sistema asigna y podremos reutilizar el codigo que tenemos que sería compatrible con este formato, pero ahora sin tener que preocuparnos de los IDs que el sistema ahora se encargará de asignar.
